## Что сделать вам

### 1. Настроить Git (один раз)

В терминале:

```bash
git config --global user.email "ваш@email.com"
git config --global user.name "Ваше Имя"
```

### 2. Создать коммит

Перейдите в папку, где лежит скрытая папка `.git` (скорее всего это **6256-main**, т.е. родительская к **bureau-app**):

```bash
cd путь\к\6256-main
git commit -m "Initial commit: bureau-experts platform"
git branch -M main
```

### 3. Создать репозиторий на GitHub

На [github.com](https://github.com) → **New repository** → имя (например, `bureau-experts`) → **Create repository** (без README).

### 4. Привязать и отправить код

Подставьте свой логин и имя репозитория:

```bash
git remote add origin https://github.com/ВАШ_ЛОГИН/ИМЯ_РЕПОЗИТОРИЯ.git
git push -u origin main
```

После этого проект будет на GitHub по ссылке вида:  
`https://github.com/ВАШ_ЛОГИН/ИМЯ_РЕПОЗИТОРИЯ`

### 5. Открыть приложение по ссылке (опционально)

- Зайдите на [vercel.com](https://vercel.com), войдите через GitHub.
- **Add New** → **Project** → выберите созданный репозиторий.
- Если в репозитории в корне лежит папка **6256-main**, в настройках проекта укажите **Root Directory**: `6256-main/bureau-app`.
- **Deploy**. Через 1–2 минуты появится ссылка вида `https://ваш-проект.vercel.app` — по ней можно открыть приложение в браузере.

Подробности — в **GITHUB_И_ДЕПЛОЙ.md**.
